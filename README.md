## 🚀 About the Project

**IDEATHON** is the official event website for the *Agentic AI Innovation Challenge* — a flagship competition under **TEXPERIA' 2026**, held at **SNS College of Technology**. This fully responsive, single-page web app delivers an immersive, cinematic experience for participants, featuring:

- ✨ Interactive 3D hero section with mouse-parallax effects
- 🌌 Animated starfield & glowing orb backgrounds
- 📋 Event details, highlights, registration, and contact sections
- 🎯 Direct integration with Google Forms registration

## ✨ Features

### 🎬 Hero Section
- Full-viewport cinematic landing with a **3D tilt effect** driven by mouse movement
- Animated **CSS starfield** (40 twinkling stars) for a space-themed ambience
- Parallax floating **glow orbs** that track cursor position
- Spring-animated headline with `framer-motion`
- Gradient CTA button + glowing gold prize badge

### 📋 Event Details
- Clean info cards: Date, Mode, Team Size & Time
- Glassmorphism card design with hover scale effects

### 💡 Why Participate? (Highlights)
- Scroll-triggered reveal animations on four key highlight cards
- Domain badge: **Agentic AI**
- Benefits: Emerging Field · Collaboration · Expert Feedback · Real-World Impact

### 📝 Registration
- Pulse-glowing registration CTA linked directly to Google Forms
- Registration fee and team details clearly laid out

### 📞 Contact
- Faculty & Student Coordinator cards with clickable phone links

## ⚡ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) `>= 18`
- [npm](https://www.npmjs.com/) or [bun](https://bun.sh/)

### Installation

```bash
# Clone the repository
git clone https://github.com/your-username/ideathon.git
cd ideathon

# Install dependencies
npm install
# or
bun install
```

### Running Locally

```bash
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173) in your browser.

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

### Run Tests

```bash
npm run test
```

## 🎨 Design System

The UI uses a **dark space-themed palette** with vibrant neon accents:

| Token | Color | Use |
|---|---|---|
| `primary` | Purple `hsl(280 80% 55%)` | Main CTAs, accents |
| `secondary` | Pink `hsl(320 70% 50%)` | Highlights, icons |
| `accent` | Rose `hsl(340 80% 50%)` | Gradient tail |
| `background` | Deep Space `hsl(270 50% 8%)` | Page background |
| `gold` | Gold `hsl(45 90% 62%)` | Prize badge |

Custom utilities used across components:
- `.gradient-text` — Purple → pink gradient text
- `.glass` — Glassmorphism card surface
- `.text-glow` / `.text-glow-sm` — Neon text glow
- `twinkle` keyframe animation — Star twinkling effect
